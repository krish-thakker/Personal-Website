# Personal Website

This is my personal portfolio website designed to showcase my professional experience, projects, skills, and resume. It is built as a modern React application using Tailwind CSS for styling.

## Features

-   **Modern UI/UX**: Dark mode design with glass-morphism effects and smooth animations.
-   **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
-   **Experience Timeline**: Interactive vertical timeline displaying professional history.
-   **Project Showcase**: Visual cards with hover effects to highlight portfolio projects.
-   **Skills Visualization**: Categorized tags for languages, frameworks, and tools.
-   **Social Integration**: Quick links to LinkedIn, GitHub, Email, etc.

## Getting Started

To run this project locally, follow these steps:

### Prerequisites
-   Node.js installed on your machine.

### Installation & Run

1.  Clone the repository and navigate to the root directory.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm start
    ```
4.  Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
